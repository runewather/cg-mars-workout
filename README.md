# CG Mars Workout

A [NextJS](https://nextjs.org/) workout app using [Material UI 5](https://mui.com/), [Jest](https://jestjs.io/) and [React Testing Library](https://testing-library.com/).

## Run app

    npm run dev
## Run tests

    npm test
